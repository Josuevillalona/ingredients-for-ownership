# Food Database Specification

## Overview
Complete specification for the Ingredients for Ownership food database containing 200+ food items organized into 8 main categories. This database forms the foundation of the ingredient recommendation system.

---

## Implementation Notes

### Data Structure
```typescript
interface FoodCategory {
  id: string
  name: string
  order: number
  description?: string
  foods: Food[]
}

interface Food {
  id: string
  name: string
  categoryId: string
  notes?: string
  servingSize?: string
  nutritionalHighlights?: string[]
  warnings?: string[] // e.g., "high mercury"
}
```

### Color-Coding Guidelines
Each food item can be assigned one of three colors by coaches:
- **Blue**: Nutrient-dense, unlimited consumption
- **Yellow**: Moderate portions, balanced intake
- **Red**: Limited portions, occasional consumption

### Search & Filtering
- Full-text search across food names
- Category filtering
- Nutritional attribute filtering (mercury content, sugar levels, etc.)
- Notes and serving size search

### Mobile Optimization
- Touch-friendly food selection
- Category-based navigation tabs
- Lazy loading for performance
- Efficient search indexing
